# 🛠️Builder

✒️Aplicação web para gerar assinaturas corporativas com customização de cores, fontes e posicionamento dos elementos na assinatura.

⚙️Tecnologias utilizadas:

* TypeScript;
* React;
* React Router;
* HTML;
* CSS;
* Bootstrap;
* React Bootstrap;
* Vite;
* SWC.

💾Comandos:

* npm i - Instalação de dependências;
* npm run dev - Execução de servidor local;
* npm run build - Criação de arquivos estáticos para produção.
